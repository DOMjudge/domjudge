\section*{Problem description}

This is a multipass-problem. In a multipass problem your submission runs multiple times for the same
testcase. For this specific problem you need to solve the game `hangman`; it should guess the right letter(s)
and solve the word(s).

\section*{Input specification}

You are given two lines;
\begin{itemize}
  \item the first containing the all guessed letters, initially empty.
  \item the second a line containing `\_` \& ` `, where `\_` must be guessed by your program
\end{itemize}
you can guess any character within the lowercase range `a-z` by outputting that one
character. You must only guess a letter once and can only guess 26 times.
